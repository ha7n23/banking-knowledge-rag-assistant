# Mobile App Access Knowledge Base

## Password Recovery

Customers who forget their mobile banking password should use the forgot password option in the mobile app. They may need to verify their registered mobile number before setting a new password.

## Registered Mobile Number Issues

If customers no longer have access to their registered mobile number, they should contact support or visit an authorised branch to update their records.

## Login Security

Customers should not share OTPs, passwords, or login credentials with anyone. Bank staff will never ask customers to disclose their password.
